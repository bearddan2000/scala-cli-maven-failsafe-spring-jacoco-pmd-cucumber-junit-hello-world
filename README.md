# scala-cli-maven-failsafe-spring-jacoco-pmd-cucumber-junit-hello-world

## Description
A POC for maven app using junit5
and cucumber framework
 with pmd,
jacoco, and failsafe plugin.

## Tech stack
- scala
- maven
	- spring
  - junit5
  - failsafe
  - jacoco
  - pmd
  - cucumber

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
